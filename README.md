# NLP_Text_Pre_Processing_Techniques

This repository demonstrates basic Natural Language Processing (NLP) text preprocessing techniques using Python and NLTK.

The project takes two paragraphs containing multiple punctuation marks and performs several preprocessing steps commonly used in NLP tasks such as text cleaning and normalization.

The implemented techniques include:

• Removing punctuation marks
• Tokenization
• Stop word removal
• Stemming using Porter Stemmer
• Lemmatization using WordNet Lemmatizer

These preprocessing steps help prepare raw text data for machine learning and deep learning models used in Natural Language Processing applications like sentiment analysis, chatbots, and text classification.

Tools and Libraries Used:
Python, NLTK

This project is part of an NLP assignment demonstrating fundamental text preprocessing techniques.
